# PyramidTeez-Website-Project

A modern React, Vite, and Tailwind CSS website for Pyramid Teez with premium black and gold branding.

## Features

- Hero section with bold calls to action
- About section
- Services section
- Custom apparel showcase
- Restaurant flyer showcase
- Contact section
- Floating WhatsApp button
- Responsive mobile-first layout

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
